# Theme Tokens

Edite `shared/theme/theme-config.ts` para customizar o sistema global.

## Cores
- `light`: palette do tema claro
- `dark`: palette do tema escuro

## Tipografia
- `typography.family*`: familias de fonte
- `typography.size*`: escala de fonte
- `typography.weight*`: pesos
- `typography.line*`: line-height
- `typography.tracking*`: letter-spacing

## Outros tokens
- `radius`: raios de borda
- `shadow`: sombras
- `spacing`: espacamentos
- `motion`: duracoes/easing de transicao

As variaveis sao injetadas globalmente pelo `ThemeStyle` no `app/layout.tsx`.
